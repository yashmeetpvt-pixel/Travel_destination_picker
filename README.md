# Travel Destination Picker

## Project Description

The Travel Destination Picker is a web application that helps users discover travel destinations based on their preferences such as budget and weather. It uses real-time data from public APIs to suggest countries, display relevant information, and allow users to explore and save destinations for future travel planning.

---

## Objective

To build an interactive and responsive web application that demonstrates API integration and the use of JavaScript array higher-order functions such as map, filter, and sort.

---

## Features

* Browse countries with detailed information
* Search destinations by country name
* Filter by region (Asia, Europe, etc.)
* Filter based on weather preferences (hot/cold approximation)
* Sort destinations by:

  * Population
  * Cost of living (approximation)
* Random Destination Generator
* Save destinations to wishlist (using localStorage)
* Fully responsive design (mobile, tablet, desktop)

---

## APIs Used

### 1. REST Countries API

Provides country details such as name, population, region, and flags
https://restcountries.com/

### 2. Unsplash API

Provides high-quality images for destinations
https://unsplash.com/developers
