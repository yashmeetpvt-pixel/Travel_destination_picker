#Travel Destination Picker

#Project Description

The Travel Destination Picker is a web application that helps users discover travel destinations based on their preferences such as region, climate, and budget. It fetches real-time country data from APIs, processes it, and displays meaningful travel insights. Users can explore destinations, filter results, and even get a random travel suggestion.

#Objective

To build an interactive and responsive web application that demonstrates API integration, DOM manipulation, and the use of JavaScript concepts such as async/await and array higher-order functions like map, filter, and sort.

#Features
Browse countries with detailed information
Search destinations by country name
Filter by region (Asia, Europe, Africa, etc.)
Filter based on climate (Tropical, Cold, Temperate, etc.)
Filter based on budget (Budget-friendly, Mid-range, Premium)
Sort destinations by:
  Name (A–Z)
  Population (ascending/descending)
  Affordability
Random Destination Generator
Dark/Light Mode toggle (stored using localStorage)
Dynamic images (with fallback if API not used)
Fully responsive design (mobile, tablet, desktop)

#APIs Used
1. REST Countries API

Provides country details such as name, population, region, capital, languages, and flags
https://restcountries.com/

2. Unsplash API

Provides high-quality images for destinations (optional integration)
https://unsplash.com/developers
